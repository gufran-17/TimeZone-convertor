Time Zone Converter
A simple **Node.js web application** that allows users to easily convert time between different cities and time zones across the world.

# Features
- Convert time from one timezone to another  
- Displays current time for selected zones  
- User-friendly and responsive interface  
- Built using **Node.js**, **Express.js**, and **HTML/CSS**



#Tech Stack
- **Backend:** Node.js, Express.js  
- **Frontend:** HTML, CSS, JavaScript  
- **Deployment:** Localhost (you can also deploy on Render/Netlify/Vercel)
